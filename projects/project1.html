DASH: A real time interview platform

It is real time coding platform to conduct online coding interviews with procturing functionalities.


View project

Description


A real-time platform to conduct inteviews online.


This project aims at creating a portal for conducting online coding interviews, in a cheat proof and proctored environment. It is an online platform where candidates are invited by the company's hiring managers to participate in a live coding interview session. It enables both the interviewer and candidate to communicate, code and collaborate interactively in real-time.




Functionalities


● An online real-time text editor.

● A proctored environment.

● Facilitates video, audio and textual communication.

● An online IDE for judging the code.

● Allows one candidate and one interviewer to communicate in one session.

● Providing secure access in each session.





Tech Stack Used


● HTML/CSS

● JavaScript

● NodeJS

● ExpressJS

● Socket IO

● WebRTC

● MongoDB
